Sobre o primeiro desafio de projeto

O Objetivo deste primeiro desafio de projeto foi fazer uso de recursos do próprio Kotlin. É um projetinho feito inteiramente em Kotlin de uma aplicação estática rodando no WebServer Ktor. Este framework foi construído com a própria linguagem Kotlin. A aplicação é empacotada em um arquivo JAR e disponibilidada num container Docker através do docker-compose.

A estrutura inicial do projeto foi feita através do site Ktor Project Generator "

O projeto final foi colocado no repositório GitHub: https://github.com/jramosfilho27/bootcamp_tqi com o nome de "projeto01"
